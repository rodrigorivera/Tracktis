POSTing JSON data

JSON can just as easily be sent in the body of a POST or PUT request. The JSON request encoder can convert a Map, List, JavaBean-style object, or a closure.

The simplest method is to use a map or list like so:

  http.request( POST, JSON ) { req ->
      body = [name:'bob', title:'construction worker']
     
       response.success = { resp, json ->
          // response handling here
      }
  }

Thanks to Json-Lib's JsonGroovyBuilder class, we an also build the request data on-the-fly like so:

  http.request( POST, JSON ) { req ->
      body = [
        first : 'Bob',
        last : 'Builder',
        address : [
          street : '123 Some St',
          town : 'Boston',
          state : 'MA',
          zip : 12345
        ]
      ]
     
      response.success = { resp, json ->
          // response handling here
      }
  }

See the API documentation for more details.
