Documentation of APIs

Here you find the whole documentation of the provided Aloqa TraX APIs. The web service API will be released soon.

  • Ajax API
    You want to develop a location-based service and integrate it with your web site? Then take the Aloqa TraX JavaScript API and build your own location-based mash-up without needing to setup an own server or to deal with positioning and transport of location data.

  • Java API
    You want to develop your own mobile application for e.g. J2ME, Symbian or Android phones? Then have a look at the Aloqa TraX Java API, which is available as software library and provides you with an easy and powerful interface.

Frequently Asked Questions

  1. Where do I get an API key?

    Send an email to requestkey@aloqa.com containing your contact info and a short description of your service.

  2. I'm developing using Eclipse. How can I embed your Javadoc of the API?

    After adding the Aloqa Trax J2ME or Android API to Eclipse, it should be listed under "Referenced Libraries". Select the jar-file and open the properties window (right-click on the jar-file). Then select on the left side "Javadoc Location". Then set the Javadoc location path to "http://dev.aloqa.com/javadoc".
